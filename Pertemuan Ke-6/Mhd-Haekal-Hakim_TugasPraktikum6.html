Uji Box-cox

# Bikin Index
index <- seq_along(dt_rg)
# Buat Data Supaya Positif
shift_val <- abs(min(dt_rg)) + 1
dt_rg_shift <- dt_rg + shift_val
# Regresi Linear untuk Box-Cox
lm_bc <- lm(dt_rg_shift ~ index)
# Uji Box-Cox
bc <- boxcox(lm_bc, lambda = seq(0, 5, by = 0.001))

# Cari Lambda Optimum
lambda_opt <- bc$x[which.max(bc$y)]
ci_lambda <- bc$x[bc$y > max(bc$y) - 0.5 * qchisq(0.95, 1)]

cat("Lambda Optimum:", lambda_opt, "\n")

Uji Formal

#1) Sisaan Menyebar Normal
ks.test(sisaan,"pnorm") 
## 
##  Asymptotic one-sample Kolmogorov-Smirnov test
## 
## data:  sisaan
## D = 0.49612, p-value < 2.2e-16
## alternative hypothesis: two-sided

Selain dengan eksplorasi, asumsi tersebut dapat diuji menggunakan uji formal. Pada tahapan ini uji formal yang digunakan untuk normalitas adalah uji Kolmogorov-Smirnov (KS). Hipotesis pada uji KS adalah sebagai berikut.

\(H_0\): Sisaan menyebar normal

\(H_1\): Sisaan tidak menyebar normal

Berdasarkan uji KS tersebut, didapat p-value < 0.0.5 yang lebih kecil dari taraf nyata 5% sehingga tolak \(H_0\) dan menandakan bahwa sisaan menyebar tidak normal. Hal ini sesuai dengan hasil eksplorasi menggunakan plot kuantil-kuantil normal.

plot(ramalan)

# Inverse differencing
# xi harus berupa nilai (numeric) terakhir dari data ORIGINAL sebelum differencing
xi_value <- as.numeric(tail(train.ts, 1))   # length = 1 karena differences = 1

# gunakan nama variabel yang tidak menimpa fungsi "forecast"
forecast_inv <- diffinv(data.ramalan, differences = 1, xi = xi_value)

# diffinv mengembalikan panjang = length(data.ramalan) + 1, sehingga buang elemen pertama
hasil <- as.numeric(forecast_inv[-1])

# pastikan panjang sama dengan test set
hasil <- hasil[seq_len(length(test.ts))]

# Bandingkan aktual vs forecast
perbandingan <- cbind(
  Aktual   = as.numeric(test.ts),
  Forecast = hasil
